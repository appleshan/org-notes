1.查看系统内核版本命令：

# uname -a

# more /etc/*release

# more /etc/issue

# more /proc/version


2.查看 cpu 信息

# grep "model name" /proc/cpuinfo

# more /proc/cpuinfo


3.查看 cpu 的位数

# getconf LONG_BIT

# ls /
要是在 root下[ls /]有 lib64 文件夹说明系统是 64


4.查看内存信息

# more /proc/meminfo

# grep MemTotal /proc/meminfo


5.查看libc、gcc 版本

# ldd /sbin/mii-tool

# rpm -qa | grep glibc

# gcc –version
